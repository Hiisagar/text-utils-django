# Text Utils Django

A simple **Text Utility Web Application** built using **Python and Django** while learning Django from Code With Harry.

This project allows users to analyze and modify text using different utilities like removing punctuation, converting to uppercase, removing extra spaces, and counting characters.

---

## 🚀 Features

* Remove Punctuation from text
* Convert text to **UPPERCASE**
* Remove **new line characters**
* Remove **extra spaces**
* **Character count** of the entered text

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/Hiisagar/text-utils-django.git
```

2. Go to the project directory

```
cd text-utils-django
```

3. Create virtual environment

```
python -m venv venv
```

4. Activate virtual environment

Windows:

```
venv\Scripts\activate
```

5. Install Django

```
pip install django
```

6. Run the development server

```
python manage.py runserver
```

Open in browser:

```
http://127.0.0.1:8000
```
